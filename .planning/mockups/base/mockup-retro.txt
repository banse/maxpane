NOTES / LEGEND
════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

TERMINOLOGY MAPPING:
  Token Price      -> POWER           (How strong the fighter hits)
  24h Volume       -> WAR CHEST       (Resources accumulated in battle)
  Momentum Rating  -> STRENGTH ★      (Star rating 1-5 based on multi-timeframe momentum)
  Momentum Score   -> LEVEL (LV.XX)   (1-99 scale mapped from momentum indicators)
  Top Gainers      -> RISING CHAMPIONS
  Top Losers       -> FALLEN WARRIORS
  Trending Pools   -> ARENA BATTLES
  Recent Txns      -> QUEST LOG
  Network Stats    -> GUILD STATS
  Big price move   -> CRITICAL HIT / KNOCKOUT / BOSS MODE

COLOR SCHEME:
  [GRN] = Positive price action, buys, rising champions
  [RED] = Negative price action, sells, fallen warriors
  [AMB] = Neutral / labels / timestamps
  [CYN] = Headers, titles, network info
  [YEL] = Star ratings (high), hotkeys, ranks
  [MAG] = Levels (LV.XX), special events
  [WHT] = Default text, token names, prices

STAR RATING CRITERIA (STRENGTH):
  ★☆☆☆☆  = Bearish all timeframes (momentum score 1-19)
  ★★☆☆☆  = Mostly bearish, slight recovery (momentum score 20-39)
  ★★★☆☆  = Mixed / neutral momentum (momentum score 40-59)
  ★★★★☆  = Bullish most timeframes (momentum score 60-79)
  ★★★★★  = Bullish all timeframes, strong trend (momentum score 80-99)

POWER BAR RENDERING:
  ████████████████  = Full bar (LV.99)
  ████████░░░░░░░░  = Half bar (~LV.50)
  ████░░░░░░░░░░░░  = Low bar (~LV.25)
  Bar width: 16 chars (█ for filled, ░ for empty)

DATA SOURCES:
  - GeckoTerminal API: Token prices, pool data, trending pools
  - Base RPC: Block number, gas price
  - Momentum calculated from 5m, 1h, 4h, 24h price changes

KEYBOARD CONTROLS:
  [Q]     = Quit application
  [R]     = Force refresh all data
  [T]     = Cycle color theme (retro/neon/dark/light)
  [TAB]   = Toggle game mode overlay (animations, sound effects reference)
  [1-5]   = Switch between arena views (different token lists / filters)
