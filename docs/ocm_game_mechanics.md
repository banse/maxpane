# Onchain Monsters (OCM) - Game Mechanics

## Overview

On-Chain Monsters is a **100% on-chain**, community-owned NFT collection on **Ethereum mainnet**. Up to **10,000 unique monsters** are randomly generated by the smart contract using geometry data (quad curves and lines) stored directly on-chain. There is no IPFS, no external API, no centralized server -- the artwork is generated on the fly by the contract and will persist as long as Ethereum exists.

The project launched in **December 2021** as a free-to-mint, zero-royalty, community-owned collection. Contract ownership has been **burned** -- fully community-owned.

**monsterfactory.lol** is a third-party, for-fun interface to mint, stake, and manage monsters.

## Core Loop

```
Stake Monsters → Earn $OCMD → Mint More Monsters → Stake More → ...
```

## Minting

**Supply:** Up to 10,000 unique monsters (currently ~4,084 minted).

### Pricing Tiers (by supply)

| Monster Numbers | Cost |
|-----------------|------|
| #0 -- #1999 | Free (gas only) |
| #2000 -- #3999 | 1 $OCMD |
| #4000 -- #5999 | 2 $OCMD |
| #6000 -- #7999 | 3 $OCMD |
| #8000 -- #9999 | 4 $OCMD |

### Special Monsters

- Five guaranteed **golden monsters** at positions #0, #2000, #4000, #6000, #8000
- 0.01% chance to mint a golden monster at any other number
- Other super-rare traits exist

### Sacrifice/Burn Mechanic

You can sacrifice (burn) a monster to instantly mint a new one at no cost (while under 10,000 total minted), destroying the original permanently.

## Staking

- **Lock** monsters into the staking contract to generate $OCMD
- **Rate:** 1 monster staked = **1 $OCMD per day**
- **Internal rate:** `11,574,070,000,000` wei per second per monster
- **Maximum:** 10 monsters can be staked per wallet simultaneously
- **Unstaking:** Anytime; accumulated $OCMD is automatically minted on unstake
- **Claim deadline:** February 1, 2022 (`CLAIM_END_TIME`), but rewards recoverable through unstaking

## $OCMD Token (MonsterDough)

- **Type:** ERC-20 with burning capabilities
- **Decimals:** 18
- **Purpose:** Used to mint monsters beyond the free tier (#2000+)
- **Earned by:** Staking OCM NFTs
- **Emission:** 1 $OCMD per staked monster per day

## Faucet (monsterfactory.lol)

The Monster Factory frontend adds a faucet contract with:
- **Welcome mint:** 1 free mint per address (gas only)
- **Batch minting:** Mint 1-10 monsters per transaction using $OCMD
- **Approval cap:** UI caps approval at 80,000 $OCMD (not unlimited)
- Uses temporary helper contracts (TempMinter, TempImmolator) deployed via CREATE2

## Key Characteristics

- **Fully decentralized:** No external dependencies, no IPFS, no API server
- **On-chain generative art:** Vector-based (quad curves and lines), not pixel-based
- **Ownership burned:** No team control, fully community-owned
- **Zero royalty:** No marketplace fees
- **Low activity/long-tail:** Revival project with small but dedicated community
